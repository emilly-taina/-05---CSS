# Café Boêmio — Atividade 05

**Nome:** Emilly Tainá da Silva Alves

## Descrição da Atividade

Desenvolvimento de um site com no mínimo 4 páginas em HTML e CSS puro, sobre o tema de uma cafeteria fictícia chamada **Café Boêmio**, localizada em Salinas — MG.

O projeto foi desenvolvido como parte da **Atividade 05** da disciplina de Desenvolvimento Web.
